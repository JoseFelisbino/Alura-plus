# 🎬 Alura+ Landing Page

Este projeto é uma **landing page inspirada na plataforma Alura+**, desenvolvida com foco em **layout moderno, responsividade e boas práticas de HTML e CSS**.

---

## 🚀 Demonstração

👉 Acesse o projeto:
https://aluraplus-gamma-plum-75.vercel.app/

---

## 🧩 Funcionalidades

* Página inicial com destaque do produto (Combo+)
* Seções informativas sobre a plataforma
* Layout responsivo para diferentes dispositivos
* Botões de ação (CTA)
* Rodapé completo com links institucionais

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* Google Fonts (Inter)

---

## 📱 Responsividade

O projeto foi desenvolvido pensando em diferentes tamanhos de tela:

* 📺 TV
* 💻 Desktop
* 📱 Mobile

---

## 🎯 Objetivo do projeto

Este projeto foi criado com o objetivo de:

* Praticar estruturação com HTML semântico
* Aplicar conceitos de CSS Grid e Flexbox
* Trabalhar com variáveis CSS
* Desenvolver layouts modernos e organizados
